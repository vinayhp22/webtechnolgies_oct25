Humans: Skeleton(Structure of the body) - HTML(Structure) Clothes/Makeup - CSS(Styling) Nervous System or Musles - JS(actions or movements) HTML - gives structure to webpage CSS - gives styling to webpage JS - gives behaviour(how to react for user actions)

Javascript (JS) is high level, interpreted, scripting language that runs on the browser.

ReactJS, AngularJS, VueJS, NodeJS(for both backend as well as frontend) etc..

JS was created by Brenden Eich in 1995 while working at Netscape.

Why JS is important in webpages:

1. Form Validations - if user give wrong input I should validate in frontend

1st layer - frontend 2nd layer - backend 3rd layer - database In all 3 layers we need to validations. To avoid server traffic, we need to check using JS.

2. Interactive Elements

Menus, Dropdowns, Carosels(sliders), image galleries, FAQ's - Accordions

3. Handling User Events

JS listens to events like click, hover, keydown, submit etc

Ex: A search bar that results as we type.

4. Dynamic content Change

Content changes without reloading the webpage.

chatting(Whatsapp Web), live cricket scores, stock market updates etc

This is done by AJAX or Fetch API

5. Enhancing user experiance:

Animations, smooth transitions, progess bar..
