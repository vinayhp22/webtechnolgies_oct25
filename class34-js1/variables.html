Variables in JS

A variable is like a container used to store data / values.

Instead of using the actual value everywhere, we store it in a variable and reuse it.

3 types of variable

1. var - Old way & Function Scoped

Introduced in early JS

Function Scoped(not block-scoped)

Can be redeclared & updated

Hoisted(moved to top of scope) -> causes bugs

2. let - Modern Way & Block Scoped

Introduced in ES6(ECMAScript - 6) - 2015

Block-scoped(not function scoped)

can't be redeclared

Can be updated

Hoisted(moved to top of scope) -> can't able to access without assigning a value because it will go Temporary Dead Zone(TDZ)

3. const - Modern Way, Constant, Block Scoped

Introduced in ES6(ECMAScript - 6) - 2015

Block-scoped(not function scoped)

can't be redeclared

Can't be updated as well

Must be initialized at the time of declaration.

Hoisted(moved to top of scope) -> can't able to access without assigning a value because it will go Temporary Dead Zone(TDZ)
