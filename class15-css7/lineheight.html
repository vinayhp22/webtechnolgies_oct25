Line-height : spacing between two lines

It is unitless, but it multiplies with font size.

This is applied only for texts, not for images or other elements

lh10 - This is P tag with line-height 10 where font size is 18px, so total height of this element is 180px only if it is in single line.

lh1 - This is P tag with line-height 1 where font-size is 36px, so total height of this element is 36px only if it is in single line

lh2 - This is P tag with line-height 1 where font-size is un defined for this tag, so total height of this element is 32px, i.e multiplied with root element (html tag)'s default size of 16px only if it is in single line

Note: If lines increses, total height of the element also increses.

if this is in single line then total height would be 32px

if this is in 2 lines then total height would be 64px

if this is in 3 lines then total height would be 96px

if this is in 4 lines then total height would be 128px
