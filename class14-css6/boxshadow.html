Box Shadow

Adds shadow effect around an element

Can add multiple shadows by comma-separating values

If a negative offset-x - then shadow effect moves to left

If a negative offset-y - then shadow effect moves to top

Syntax:

box-shadow: offset-x offset-y blur-radius spread-radius color
