2. Hexadecimal(HEX) - most used in the industry

Allowed characters/digits: 0-9, A-F/a-f

Stands for Red, Green, Blue. Values from 0 to 255

4. RGBA

Same as RGB, but with alpha transaparency(0.0 -> fully transparent, 1.0 -> fully opaque)

This is red text (rgba(255, 0, 0, 1.0))

Hue = degree(0-360)

Saturation = percentage of color intensity

Lightness = percentage of brightness
