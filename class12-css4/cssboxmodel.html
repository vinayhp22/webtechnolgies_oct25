CSS BOX MODEL

Every HTML element is treated as a rectangular box.

The box model defines how content, padding, border & margin work togerther to determine the space an element occupies on a webpage.

Every HTML element is treated as a box. CSS controls size & spacing of these boxes using Box Model.
