DOM(Document Object Model)

DOM represents the structure of a web page as Tree of objects.

When a webpage loads, the browser creates a DOM that JS can interact with.

With DOM, we can add, change or remove HTML elements & CSS styles.

Example:

HTML: a blueprint of house

DOM: Actual house built from the blueprint(can now be modified, decorated, repainted).
