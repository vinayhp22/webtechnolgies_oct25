Event Handling in JS

An event is a action that performed by a user(click, hover, change, typing)

others: onkeydown, onkeyup, onload, onsubmit, ondbclick

Inline events vs Event Listners

i). Inline Events

Written directly in HTML(inside tag)

Not recomended in modern coding(mixes HTML with JS)

ii). Event Listeners

Modern Way -> keeps HTML, JS separate

Use addEventListener() method

Advantages of Event Listeners

1. Can attach multiple events to the same element

Keeps HTML & JS separate(clean code)

More flexible & maintainable
