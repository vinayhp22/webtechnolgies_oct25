Bootstrap

A Front-end framework for building responsive and mobile first websites quickly.

Created by Twitter Developers( now it's X), it combines HTML, CSS & JS components to make modern UI(User Interface) design easier.

Key Features

  1. Responsive design out of the box - adapts to all devices
  2. Pre-styled components - buttons, forms, navbars, modals, etc
  3. Consistent design - same look across browsers and platforms
  4. Fast Development - less time writing custum CSS
  5. Community Support - Huge documentation and eco-system
With Bootstrap

Adding Bootstrap - 2 ways

  • Using a CDN(Content Delivery Network)..
    1. Quick Setup
    2. Requires Internet
    3. Sometimes CDN Server down - your website will load just HTML, no CSS styling
    4. Automatic caching: page loads faster
  • Dowloding & hosting locally..
    1. Works Offline
    2. Full Control over the version
