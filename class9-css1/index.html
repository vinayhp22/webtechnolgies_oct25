HTML - Hyper Text Markup Language - Structure(Content)

- Structure for a web page.

CSS - Cascading Style Sheet - Styling(Presentation)

- is used to make Styling for web page that build on HTML.

Examples:

HTML - Human Skeleton,
CSS - MakeUp, Clothes, Shoes,

Types of CSS

1. inline CSS

Inline CSS is written inside the HTML element's style attribute.

It affects only the specific element.

This is p tag for inline CSS

This is another p tag for inline CSS

2. Internal CSS

Internal CSS is written inside the style tag within the head section of the HTML page.

It is used to style a single web page.

3. External CSS

Exteranl CSS is written in a separate .css file, linked to HTML using link tag.
