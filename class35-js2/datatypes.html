Datatypes in JS

Datatype = tells us what is the type of data that a variable can hold.

Java - statically typed language

Python - dynamically typed language

JS is dynamically typed language - data type determined at runtime.

Data Types - Types

1. Primitive Data Types - predefined data types

1. String

2. Number

3. Boolean

4. Undefined - Declared but not assigned

5. Null - intentional empty value

6. Symbol - unique identifier

7. BigInt(ES2020) - very large Integers

2. Non-primitive Data types - not predefined

1. Objects

2. Arrays

Primitive Data types

Stores value directly in memory

Immutable - can't change the data

Ex: Strings, numbers etc

Non-Primitive Data types or Reference data types

Stores only references(memory address) in memory

Muttable - can change the data

Ex: Objects, Arrays
